Project Script Guide:

- Setup Application
  - `yarn install`
- Run Development Environment
  - `yarn rock:dev`
- Run Storybook
  - `yarn rock:storybook`
- Run Tests
  - `yarn rock:test`
  - `yarn rock:cart:test`
  - `yarn rock:product:test`
- View Code Coverage of the Application
  - `yarn rock:cc`
  - `yarn rock:cart:cc`
  - `yarn rock:product:cc`
- Run Lint and Prettier
  - `yarn rock:lint`
  - `yarn rock:format`
